# Post-ASI Ethos Framework

This is the linear operating path for a person living in a post-ASI world.

The order is strict: stabilize self first, then stabilize close relationships, then take societal responsibility, then steward the long future.

## Core Rule

At every phase, apply the same four commitments:

1. Purpose: act intentionally.
2. Integrity: align values, words, and behavior.
3. Long-term responsibility: optimize for durable outcomes.
4. Contribution: improve self, relationships, society, and future generations.

## Operating Cadence

Use this cadence while moving through phases:

1. Daily: one behavior from the current pillar.
2. Weekly: one measurable correction.
3. Monthly: one output proving progress.
4. Quarterly: phase readiness check and advancement decision.

In a post-ASI context, capability can scale fast. Character must scale faster.
